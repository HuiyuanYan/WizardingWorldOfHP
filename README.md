# 三、构建手册
## 1. 本地单机伪分布式
搭建好上述环境后，进入项目目录下，运行
```shell
mvn clean install
```
下载所需依赖。

```shell
mvn clean package
```
在target目录下生成jar包。

启动本机Hadoop环境，上传输入文件，运行命令：
```shell
hadoop jar XXX.jar [arg1] [arg2]...
```
运行jar包。
